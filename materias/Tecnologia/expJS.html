Os Músculos do Navegador

Se o HTML estrutura o conteúdo e o CSS define o visual, o JavaScript (JS) é a real linguagem de programação que roda diretamente no navegador do usuário. Sua função principal é dar vida à página, controlando comportamentos dinâmicos, respondendo a cliques e processando dados em tempo real.

1. Qual a função principal do JavaScript?

A missão do JavaScript é reagir a eventos e modificar a página web sem que o usuário precise recarregá-la.

Manipulação do DOM

O JavaScript consegue acessar a estrutura do HTML (chamada de DOM) e alterá-la dinamicamente. Ele pode criar novos textos, alterar cores que vieram do CSS, esconder ou exibir elementos na tela quando o usuário realiza uma ação.

Resposta a Eventos

O JavaScript fica "escutando" as interações do usuário. Exemplos comuns de gatilhos controlados pelo JS:

  • O clique em um botão.
  • O envio de um formulário de cadastro.
  • O movimento do mouse ou a rolagem da página.

2. Como a lógica é aplicada?

Diferente do HTML e do CSS, no JavaScript usamos recursos de programação reais como variáveis, funções e tomada de decisões.

Exemplo Prático: Criando um Alerta

Veja como o JavaScript usa funções e eventos para capturar um clique e exibir uma mensagem interativa na tela:

// 1. Seleciona o botão no HTML usando seu ID
const botao = document.getElementById('meuBotao');

// 2. Cria uma função para o comportamento do clique
function exibirMensagem() {
  alert('Você clicou no botão e o JavaScript funcionou!');
}

// 3. Avisa ao botão para escutar o evento de "click"
botao.addEventListener('click', exibirMensagem);
