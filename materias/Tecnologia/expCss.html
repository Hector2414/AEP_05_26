A Roupa do HTML

Se o HTML é o esqueleto de uma página web, o CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é a pele, a roupa e a maquiagem. Ele é uma linguagem de estilo utilizada para definir a apresentação visual de documentos web, controlando cores, fontes, layouts, espaçamentos e responsividade.

1. Como o CSS funciona?

O CSS funciona aplicando regras visuais a elementos específicos do HTML através de uma sintaxe simples e direta.

Anatomia de uma Regra

Uma regra CSS é composta por um Seletor (o elemento que será modificado) e um bloco de declarações contendo Propriedades e Valores.

p {
  color: blue;
  font-size: 16px;
}
O que é a Cascata?

O termo "Cascata" significa que as regras de estilo são aplicadas de cima para baixo. Se houver duas regras conflitantes para o mesmo elemento, o navegador geralmente aplicará a que foi declarada por último ou a que tiver maior especificidade.

2. Formas de Aplicar CSS

Existem três maneiras de incluir estilos CSS em um documento HTML, cada uma adequada para diferentes situações.

CSS Inline e Interno

Inline: Aplicado direto na tag HTML usando o atributo style. Ruim para manutenção.

<p style="color:red;">Texto</p>

Interno: Escrito dentro da tag <style> no cabeçalho do HTML (exatamente como esta página foi feita).

CSS Externo (Recomendado)

Os estilos são salvos em um arquivo separado com a extensão .css e vinculados ao HTML através da tag <link> dentro do head.

<link rel="stylesheet" href="estilos.css">

Permite alterar o visual de centenas de páginas modificando um único arquivo.
