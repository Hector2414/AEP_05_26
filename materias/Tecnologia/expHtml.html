O Esqueleto da Web

O HTML (Linguagem de Marcação de Hipertexto) não é uma linguagem de programação, mas sim uma linguagem de marcação. Ela serve para definir a estrutura e o conteúdo de uma página web, indicando ao navegador o que é um título, o que é um parágrafo, onde ficam as imagens e como os links se conectam.

1. Como o HTML funciona?

A base do HTML baseia-se em elementos encapsulados por etiquetas, conhecidas popularmente como tags.

Anatomia de uma Tag

A maioria das tags possui uma abertura, o conteúdo interno e um fechamento sinalizado com uma barra (/).

<p>Isto é um parágrafo.</p>

O navegador interpreta essa instrução e exibe apenas o texto de forma limpa na tela.

Marcação Semântica

Tags semânticas dão significado ao conteúdo, ajudando os motores de busca (como o Google) e os leitores de tela a entenderem a função de cada parte do site.

  • <header>: Cabeçalho da página.
  • <main>: Conteúdo principal.
  • <footer>: Rodapé do site.

2. A Estrutura Essencial

Todo arquivo HTML válido precisa seguir uma árvore de dependências padrão para ser reconhecido corretamente pelos navegadores.

O Modelo Padrão (Template)

Veja a hierarquia mínima de códigos necessária para começar qualquer página na internet:

<!DOCTYPE html> <!-- Avisa ao navegador que é HTML5 -->
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8"> <!-- Permite acentos -->
    <title>Título da Aba</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1> <!-- Conteúdo visível -->
  </body>
</html>
