Raciocínio Lógico

A ponte de ouro entre a Matemática Pura e a Ciência da Computação

O Elo Perdido

O raciocínio lógico é o que permite transformar o pensamento humano em instruções que uma máquina consegue processar. Na matemática, usamos proposições e conectivos para julgar o que é Verdadeiro ou Falso. Na computação, esse sistema se torna a Álgebra Booleana, onde Verdadeiro vira 1 e Falso vira 0 (os famosos bits).

1. Operadores Lógicos (Matemática)

Estruturas usadas para combinar proposições simples e formar frases complexas.

Conjunção (E / ∧)

Só resulta em Verdadeiro se todas as proposições envolvidas forem verdadeiras ao mesmo tempo.

ABA ∧ B
VVV
VFF
FVF
FFF
Disjunção (OU / ∨)

Resulta em Verdadeiro se pelo menos uma das proposições envolvidas for verdadeira.

ABA ∨ B
VVV
VFV
FVV
FFF

2. A Aplicação Prática na Computação

Como os softwares e hardwares utilizam essas regras matemáticas para tomar decisões e funcionar.

Hardwares: Portas Lógicas

Os processadores de computadores e celulares contêm bilhões de transistores que agem como interruptores físicos baseados na lógica booleana.

  • A tabela do "E" vira a porta física AND.
  • A tabela do "OU" vira a porta física OR.
  • A negação vira a porta NOT (inverte 0 para 1 e vice-versa).
Softwares: Condicionais (Se / If)

Qualquer programa de computador toma decisões utilizando os conectivos lógicos da matemática em blocos condicionais.

Exemplo em código:
if (usuarioLogado == true && senhaCorreta == true) {
  // Permite o acesso
} else {
  // Bloqueia o acesso
}
